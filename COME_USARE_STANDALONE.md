# Se non hai scaricato lo ZIP e non hai i file

Se non riesci a scaricare `release/il-faro-canicatti-portale.zip`, usa la versione in un solo file:

```text
PORTALE_STANDALONE.html
```

## Metodo più semplice

1. Scarica o copia solo il file `PORTALE_STANDALONE.html`.
2. Salvalo sul Desktop.
3. Fai doppio click su `PORTALE_STANDALONE.html`.
4. Il portale si apre nel browser.

Questa versione non richiede:

- ZIP
- Git Bash
- npm
- Node.js
- Python
- server locale

## Se non riesci a scaricare nemmeno il file HTML

1. Apri il file `PORTALE_STANDALONE.html` nel repository online.
2. Clicca su **Raw**.
3. Seleziona tutto il contenuto con `CTRL+A`.
4. Copia con `CTRL+C`.
5. Apri Blocco Note sul tuo PC.
6. Incolla con `CTRL+V`.
7. Salva il file con questo nome esatto:

```text
PORTALE_STANDALONE.html
```

8. Fai doppio click sul file salvato.

## Differenza rispetto alla versione completa

La versione standalone serve per vedere subito l'anteprima. Contiene HTML, CSS, logo e JavaScript nello stesso file.

La versione completa del progetto resta organizzata in più file ed è quella da usare quando passeremo a un vero repository grande con backend, database, login utenti e area amministrativa reale.
